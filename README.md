# car-app-scraping

## Installation

######car-app-scraping APP

Install dependecies and devDependencies

```sh
npm i
```

Enter the following command

```sh
npm run dev
```

###### Backend/server

Install dependecies and devDependencies

```sh
npm i
```

Enter the following command

```sh
node server.js
```

###### Frontend/client

```sh
`cd client` go into the folder
```

Install dependecies and devDependencies

```sh
npm i
```

Enter the following command

```sh
npm start
```
