# car-app-scraping

## Installation

###### Backend

Install dependecies and devDependencies

```sh
npm i
```

Enter the following command

```sh
node server.js
```

###### Frontend

```sh
`cd client` go into the folder
```

Install dependecies and devDependencies

```sh
npm i
```

Enter the following command

```sh
npm start
```
